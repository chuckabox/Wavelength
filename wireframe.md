## Key Sections

| Section | Purpose |
|---|---|
| **Header** | App name, settings, mode toggle (Live/Upload) |
| **Live Camera Feed** | Webcam stream with Start/Upload/Practice Mode controls |
| **Detected Face + Confidence** | Shows who's being analyzed and confidence level |
| **Interpretation** | The reasoning text — the core product value |
| **Signals + Audio** | Visual emotion bars + audio cues (tone, pace, volume) |
| **Timeline** | Scrollable history of timestamped cue reads |
